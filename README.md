# zhiying-Content-Generation-Marketing
随着短视频和内容营销的爆发式增长，企业对高质量、个性化营销内容的需求急剧上升。然而，传统内容耗时多、成本高、流程复杂，难以满足顾客需求的快速变化和市场的剧烈切换。因此  本项目——智影文创，是一款面向中小企业的智能营销内容自动化生成平台。平台以大语言模型（LLM）为核心驱动，融合多Agent协同架构、RAG知识检索增强、向量语义匹配、ComfyUI数字人视频生成等前沿技术，实现从输入业务信息到输出可发布营销内容的全链路自动化。覆盖营销文案生成、营销图片生成、营销视频生成、数字人生成、热点爬取等功能，用户只需上传业务数据，平台即可自动生成营销文案、小红书图文、短视频、等多种形式的营销内容，并实现辅助分发，实时检测点击
